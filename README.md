# Bike Sales Data Analysis Project

## Overview
This project analyzes bike sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Files Included
- Bike_Sales_Data.xlsx
- Bike_Sales_Presentation.pptx
- README.md

## Key Insights
- Higher income customers purchased more bikes
- Middle-aged customers contributed most sales
- Commute distance affected purchasing behavior
